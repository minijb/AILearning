**分析日期：** 2026-04-01
**分析范围：** 新手引导流程、信息架构、界面布局和情感化设计

## 一、新手引导流程设计

### 1.1 新手引导目标

**核心目标：**
- 5分钟内让玩家理解核心循环
- 10分钟内完成第一场完整战斗
- 30分钟内获得第一个SSR武将
- 1小时内解锁所有基础系统

**引导设计原则：**
1. **即时反馈**：每个操作立即给予视觉/听觉反馈
2. **渐进披露**：不一次性展示所有功能，按需解锁
3. **降低门槛**：用自动跳过/推荐设置减少选择压力
4. **情感引导**：用故事包装教学，不干巴巴讲机制

### 1.2 新手引导流程（7日留存设计）

**第1日：核心循环建立**

```
┌─────────────────────────────────────────────────────────────┐
│ 时间线 │ 操作 │ 反馈 │ 情感设计                          │
├─────────────────────────────────────────────────────────────┤
│ 0:00 │ 启动游戏 │ 播放Logo动画 │ 品牌感知                      │
│ 0:30 │ 选择阵营 │ 魏/蜀/吴/群选择 │ "选择你的阵营"故事引导         │
│ 1:00 │ 教程战斗 │ 自动引导放置 │ "让我来教你排兵布阵"              │
│ 2:00 │ 首次普攻 │ 伤害数字飞出 │ "普通攻击，威力虽然不大..."       │
│ 3:00 │ 首次技能 │ 技能特效爆发 │ "关键时刻要靠技能扭转战局！"       │
│ 4:00 │ 通关结算 │ 3星展示+奖励 │ "首战告捷！将军威武！"          │
│ 5:00 │ 首次升级 │ 属性数字滚动 │ "恭喜升级！攻防都有提升！"        │
│ 6:00 │ 教程结束 │ 解锁全部功能 │ "基础教学完成，自由征战吧！"      │
└─────────────────────────────────────────────────────────────┘
```

**第1-3日：系统解锁**

```
┌─────────────────────────────────────────────────────────────┐
│ 日期 │ 解锁系统 │ 引导方式 │ 目标                         │
├─────────────────────────────────────────────────────────────┤
│ 第1日 │ 武将界面 │ 弹窗引导 │ 了解武将升级                      │
│ 第1日 │ 装备界面 │ 弹窗引导 │ 了解装备强化                      │
│ 第2日 │ 羁绊系统 │ 提示发现 │ "发现新的羁绊组合！"               │
│ 第2日 │ 抽卡系统 │ 新手十连 │ 首次保底SSR                      │
│ 第3日 │ 阵法系统 │ 提示解锁 │ "解锁新阵法，战力提升！"           │
│ 第3日 │ 军衔系统 │ 任务引导 │ 完成军衔任务                      │
└─────────────────────────────────────────────────────────────┘
```

**第4-7日：深度体验**

```
┌─────────────────────────────────────────────────────────────┐
│ 日期 │ 引导内容 │ 目标                                 │
├─────────────────────────────────────────────────────────────┤
│ 第4日 │ 精英关卡 │ 体验装备掉落和碎片获取                  │
│ 第5日 │ 联盟系统 │ 加入联盟，体验社交                      │
│ 第6日 │ 竞技场 │ 首次PVP，体验公平竞技                    │
│ 第7日 │ 成就系统 │ 完成首个成就，获得奖励                  │
│ 第7日 │ 无限挑战 │ 体验无尽模式                          │
└─────────────────────────────────────────────────────────────┘
```

### 1.3 引导界面设计规范

**引导高亮规范：**
- 高亮区域使用金色边框 + 脉冲动画
- 非引导区域使用半透明黑色遮罩（60%透明度）
- 引导手指使用动态PNG序列，0.3秒点击动画

**引导文案规范：**
- 每句话不超过20个字
- 使用玩家视角（"你"而非"玩家"）
- 正面鼓励为主，避免负面词汇
- 配合音效增强氛围

## 四、情感化设计要点

### 4.1 战斗反馈设计

**伤害数字反馈：**
| 攻击类型 | 数字颜色 | 数字大小 | 附加效果 |
|---------|---------|---------|---------|
| 普通攻击 | 白色 | 中 | 无 |
| 暴击 | 金色 | 大 | 屏幕微震 |
| 技能伤害 | 蓝色 | 大 | 技能特效 |
| 范围伤害 | 红色 | 中 | 数字飘散 |
| 治疗 | 绿色 | 中 | 数字上飘 |
| 击杀 | 紫色 | 最大 | 击杀动画 |

**技能释放反馈：**
```
技能按钮可释放时：
├── 按钮发光脉冲动画
├── "可释放" 标签显示
└── 震动反馈（可选）

技能释放中：
├── 技能图标放大 + 拖尾
├── 技能音效播放
├── 技能特效覆盖目标
└── 伤害/效果数字飞出

技能冷却中：
├── 按钮灰显
├── 冷却进度圆形填充动画
└── 倒计时数字显示
```

### 4.2 成长反馈设计

**升级反馈：**
```
升级触发时：
1. 角色发光（从上到下扫光）
2. 属性数字滚动增长
3. "+1" 符号从属性飞出
4. 升级音效播放
5. 经验条瞬间补满动画
```

**突破反馈：**
```
突破触发时：
1. 全屏变暗
2. 武将立绘放大居中
3. 星级动画逐个点亮
4. 突破特效（光芒四射）
5. 属性全面提升展示
6. 突破成功提示
7. 可继续突破提示（若材料足够）
```

**羁绊激活反馈：**
```
羁绊激活时：
1. 羁绊图标高亮动画
2. "羁绊激活！" 提示
3. 羁绊效果文字展示
4. 相关武将头像连线光效
```

### 4.3 音乐与音效设计

| 场景 | 背景音乐 | 音效类型 |
|-----|---------|---------|
| 主界面 | 轻快三国风BGM | 点击确认音 |
| 战斗准备 | 紧张备战BGM | 武将放置音 |
| 战斗进行 | 战斗BGM | 普攻、技能、击杀音 |
| BOSS战 | 史诗战斗BGM | 特殊打击音 |
| 通关结算 | 胜利BGM | 星级音效 |
| 抽卡 | 期待BGM | 抽卡音效、SSR警报 |
| 升级 | 成长BGM | 升级音效 |

## 五、情感曲线设计

### 5.1 单局战斗情感曲线

```
情感强度
    ↑
    │            ★ BOSS战
    │          ╱  ╲
    │        ╱      ★ 通关
    │      ╱    ★★★
    │    ╱  ★★★  ★★★ 战斗高潮
    │  ★★★
    │ ★★★ 前期
    │★★★ 铺垫
    │
    └──────────────────────────────────────────→ 时间
      0s  30s  60s  90s  120s  150s  180s
```

**设计要点：**
- 前30秒：铺垫期，小怪出现，玩家熟悉操作
- 30-90秒：积累期，怪物变强，气氛紧张
- 90-150秒：高潮期，BOSS出现，技能全开
- 150-180秒：收尾期，胜负已定，结算展示

### 5.2 新手7日情感设计

```
情感强度
    ↑
    │      ★ 第7日
    │    ╱   自由探索
    │  ★   第2-3日
    │ ╱     系统熟悉
    │★
    │      第1日
    │ ★★★   核心循环建立
    │ ★★★  首次体验峰值
    │
    └───────────────────────────────────────────→ 日期
      D1   D2   D3   D4   D5   D6   D7
```

## 六、可访问性设计

### 6.1 色盲友好

| 设计要点 | 实施方案 |
|---------|---------|
| 伤害数字颜色 | 不依赖红绿色区分，用形状+数字区分 |
| HP条颜色 | 蓝（满）-黄（中）-红（低），另加图案 |
| 怒气条颜色 | 固定金色，无颜色编码信息 |
| 阵营颜色 | 魏（蓝）/蜀（绿）/吴（红）/群（紫）+ 图标区分 |

### 6.2 界面适配

| 设备类型 | 设计方案 |
|---------|---------|
| 手机竖屏 | 默认设计，全屏适配 |
| 手机横屏 | 战斗界面横屏，UI适配 |
| 平板 | 更大战斗区域，UI间距增加 |
| PC模拟器 | 支持键鼠操作 |

### 6.3 操作辅助

| 功能 | 设计方案 |
|-----|---------|
| 自动战斗 | 一键开启，自动释放技能 |
| 跳过战斗 | 通关后可跳过（节省时间） |
| 一键升级 | 自动升级所有可升级武将 |
| 推荐阵型 | AI推荐最优阵型（新手期） |
