# 三国杀战棋游戏 — 类型架构分析报告

**分析日期：** 2026-04-01
**分析范围：** 基于循环勇者、王国保卫战、背包乱斗三款参考游戏，提取"放置战棋"品类的通用架构

---

## 一、游戏类型定位

### 1.1 新品类定义："放置战棋"(Placement Auto-Chess)

本游戏定位为**放置战棋**（Placement Auto-Chess），这是对"逆塔防放置"、"战棋自动战斗"和"卡牌技能系统"三种玩法范式的有机融合。

**品类定义公式：**
```
放置战棋 = 循环勇者（放置机制）× 战棋（格子系统）× 背包乱斗（配置即策略）× 三国杀（技能系统）
```

**核心区别于现有品类：**

| 品类 | 操控方式 | 战斗主体 | 策略焦点 | 典型游戏 |
|-----|---------|---------|---------|---------|
| 传统战棋 | 逐格操控每个单位 | 玩家完全控制 | 走位、攻击时机 | 火焰纹章、高级战争 |
| 传统塔防 | 建造防御塔 | 塔攻击敌人 | 塔位选择、升级顺序 | 王国保卫战、植物大战僵尸 |
| 自走棋 | 配置阵容后自动战斗 | 棋子在棋盘移动 | 阵容搭配、站位 | 云顶之弈、金铲铲 |
| 放置游戏 | 设置后自动挂机 | 无直接战斗操控 | 数值积累、资源分配 | 放置江湖、AFK Arena |
| **放置战棋** | **放置单位+手动技能** | **武将自动走格** | **阵位+技能+羁绊** | **本游戏** |

### 1.2 核心玩法概述

**游戏核心循环：**

```
┌──────────────────────────────────────────────────────┐
│                                                       │
│   布置阵位 ──→ 启动战斗 ──→ 自动走格 ──→ 武将攻击   │
│      ↑                          ↓                      │
│      ↑        手动技能 ◄────── 技能触发              │
│      ↑                          ↓                      │
│      ↑                      击杀敌人                  │
│      ↑                          ↓                      │
│      ↑ ←──────── 资源结算 ←── 回合结束                │
│      ↑                                               │
│   调整阵型 ──→ 升级武将 ──→ 挑战下一关               │
│                                                       │
└──────────────────────────────────────────────────────┘
```

### 2.2 L1 核心循环层详解

**战斗引擎子系统：**
- 战棋格子系统：8列 × 6行标准地图
- 自动寻路：曼哈顿距离路径算法
- 攻击判定：命中率 = 基础命中率 × 兵种克制 × 属性差
- 伤害计算：攻击 × 技能系数 × 克制系数 × 暴击加成
- 胜负判定：敌人全灭则通关，我方武将全灭则失败

**放置系统子系统：**
- 拖拽放置：鼠标/触控拖拽武将到放置区
- 阵型槽位：3×2标准阵型，最多6名武将
- 羁绊检测：实时显示激活的羁绊效果
- 武将召回：战斗中可消耗资源召回并重新放置武将

**技能系统子系统：**
- 主动技能：消耗怒气，手动释放，需选择目标
- 被动技能：条件触发，自动生效
- 终极技能：怒气满时释放，大范围高效果
- 技能冷却：使用后进入冷却，倒计时显示

**关卡系统子系统：**
- 地图加载：每关独立地图文件
- 波次管理：敌人按波次刷新，波次间隔可暂停
- 敌人AI：追击、攻击、施放技能
- 星级评价：基于通关状态给予1-3星

### 2.3 L2 成长循环层详解

**武将养成子系统：**
- 等级：经验值升级，100级上限
- 突破：6星突破系统，碎片升级
- 觉醒：5★武将觉醒，解锁第四技能
- 技能：3个技能槽位，支持升级和替换

**装备系统子系统：**
- 武器：增加攻击属性
- 防具：增加防御、生命属性
- 饰品：提供特殊被动效果
- 套装：集齐套装激活额外效果

**军衔系统子系统：**
- 军衔等级：士兵 → 校尉 → 将军 → 大将军
- 每日任务：军衔专属任务奖励
- 军衔特权：跳过战斗、额外资源等

### 2.4 L3 策略深度层详解

**天赋系统（局外全局强化）：**
- 进攻系：伤害、暴击、穿透相关
- 防御系：生命、护甲、闪避相关
- 辅助系：怒气获取、治疗效果相关
- 特殊系：解锁新机制（连杀、资源加成）

**阵法系统（阵型策略）：**
- 6种阵法：锋矢阵、偃月阵、八卦阵、长蛇阵、箕形阵、鹤翼阵
- 位置加成：不同阵法提供不同位置的效果
- 阵法升级：5级升级路径

**羁绊系统（组合策略）：**
- 武将羁绊：双人组合羁绊（50+组合）
- 阵营羁绊：魏/蜀/吴/群各有专属羁绊
- 国家羁绊：特定武将组合触发（五虎上将等）

**难度系统（挑战多样性）：**
- 5档难度：简单/普通/困难/噩梦/地狱
- 词缀系统：每日随机词缀增加变化
- 极限模式：Roguelike随机强化

### 2.5 L4 社交与商业层详解

**联盟系统：**
- 联盟副本：团队协作挑战
- 联盟商店：专属道具购买
- 联盟科技：全员属性加成

**竞技系统：**
- 排位赛：1v1/2v2自动对战
- 巅峰赛：赛季制高额奖励
- 自走对战：玩家配置对战

**赛季系统：**
- 赛季周期：2个月一赛季
- 赛季奖励：专属皮肤、特效
- 赛季重置：段位重置，武将保留

**付费系统：**
- 武将抽取：核心付费点
- 皮肤系统：外观付费，不加属性
- 战令/通行证：轻度付费加速
- 月卡/基金：稳定付费福利

## 四、差异化机会分析

### 4.1 核心差异化优势

**1. IP差异化：三國殺品牌护城河**
- 国内最成功的三国题材卡牌游戏之一
- 深厚的武将/技能/世界观积累
- 大量忠实的三国杀玩家群体
- 品牌授权明确，无版权风险

**2. 玩法差异化："放置战棋"新品类**
- 区别于传统战棋的繁琐操作
- 区别于塔防的被动防御
- 区别于自走棋的纯配置
- 填补"有策略深度的放置游戏"空白

**3. 平台差异化：移动端优先**
- 放置机制天然适配移动端碎片时间
- 技能释放提供移动端操作的爽感
- 三国杀IP用户以移动端为主

### 4.2 潜在挑战与应对

| 挑战 | 描述 | 应对策略 |
|-----|------|---------|
| 市场认知 | 放置战棋是新概念，需要教育市场 | 前期强调"懒人战棋"、"策略挂机" |
| 三国杀粉丝 | 核心粉丝可能不接受玩法改变 | 提供经典模式 + 差异化定位 |
| 数值平衡 | 多系统耦合可能导致数值膨胀 | 设置赛季重置 + 数值上限 |
| 竞争品 | 其他厂商可能复制 | 快速迭代 + IP护城河 |

### 5.2 核心设计原则

1. **策略与执行分离**：玩家决策（阵位/技能配置）与执行（自动战斗）明确分离
2. **低操作高策略**：核心乐趣在于配置和规划，而非逐格操作
3. **成长可见**：每次通关都有明确的成长反馈
4. **资源闭环**：所有资源都有明确用途，无浪费
5. **IP深度融合**：三国杀技能系统是核心差异点，不是简单皮肤
