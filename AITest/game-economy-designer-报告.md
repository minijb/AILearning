## 一、货币体系设计

### 1.1 货币类型总览

| 货币 | 图标 | 获取方式 | 主要用途 | 可付费获取 |
|-----|------|---------|---------|-----------|
| **金币** | 金锭 | 关卡、任务、出售 | 武将升级、装备强化、阵法升级 | 可（但有周上限） |
| **元宝** | 宝石 | 充值、月卡、成就 | 招募武将、商城购买、体力购买 | 主要付费货币 |
| **战功** | 剑 | PVP竞技场 | 竞技场商店兑换 | 不可 |
| **军功** | 勋 | 战役通关 | 军衔升级、军衔商店 | 不可 |
| **联盟积分** | 齿轮 | 联盟活动 | 联盟商店兑换 | 不可 |
| **声望** | 旗 | 阵营关卡 | 阵营专属武将好感奖励 | 不可 |

### 1.2 货币产出与消耗设计

**金币循环：**
```
产出端：
├── 关卡通关：普通关50-200金，Boss关300-500金
├── 每日任务：100-300金/天
├── 成就奖励：100-1000金/成就
├── 出售装备：装备价值10-50%的金币
├── 军衔俸禄：每日领取100-1000金（按军衔）
└── 竞技场奖励：每周1000-5000金

消耗端：
├── 武将升级：100金/级（随等级增加）
├── 装备强化：50金/级（随强化等级增加）
├── 阵法升级：500-5000金（按阵法等级）
├── 技能升级：200金/级
├── 天赋重置：1000金/次
└── 商店购买：每日刷新商品
```

**元宝循环：**
```
产出端：
├── 充值：直接购买
├── 月卡：每日发放
├── 战令奖励：每赛季发放
├── 成就奖励：少量
└── 运营活动：节日赠送

消耗端：
├── 武将招募：单抽120-160元宝，十连1000-1200元宝
├── 体力购买：50元宝/次，每日5次
├── 快速战斗：30元宝/次
├── 商城直购：皮肤、礼包
├── 战令升级：解锁高级战令
└── 资源购买：金币、经验药
```

## 二、成长曲线设计

### 2.1 武将升级曲线

**经验需求公式：**
```
升级所需经验 = 基础经验 × 等级 × (1 + 等级 × 0.05)

基础经验 = 100
```

| 等级区间 | 平均每级所需经验 | 升级频率建议 |
|---------|----------------|------------|
| 1-10级 | 150-200 | 新手期，频繁升级 |
| 11-30级 | 300-800 | 成长期，稳定升级 |
| 31-50级 | 1000-2000 | 中期，节奏放缓 |
| 51-70级 | 2500-4000 | 后期，节奏较慢 |
| 71-100级 | 5000-15000 | 末期，高门槛 |

**属性成长公式：**
```
每级属性增长 = 基础成长 × (1 + 突破加成)

例：赵云
基础攻击成长 = 25
基础生命成长 = 150

10级总攻击 = 25 × 10 = 250
50级总攻击 = 25 × 50 × (1 + 0.1×突破次数)
```

### 2.2 装备强化曲线

**强化经验需求：**
```
强化所需金币 = 强化等级 × 50 + 强化等级² × 5
强化所需材料 = 强化等级 × 10
```

| 强化等级 | 金币消耗 | 成功率 | 失败惩罚 |
|---------|---------|-------|---------|
| 1-10级 | 100-2000 | 100% | 无 |
| 11-20级 | 2500-10000 | 100% | 无 |
| 21-30级 | 12000-40000 | 80% | 降1级 |
| 31-40级 | 50000-150000 | 60% | 降3级 |
| 41-50级 | 180000-500000 | 40% | 降5级 |

### 2.3 阵法升级曲线

**升级消耗：**
| 阵法等级 | 金币 | 材料需求 | 升级时间 |
|---------|------|---------|---------|
| 1→2级 | 1000 | 10个基础残卷 | 即时 |
| 2→3级 | 3000 | 30个基础残卷 | 即时 |
| 3→4级 | 8000 | 10个高级残卷 | 即时 |
| 4→5级 | 20000 | 30个高级残卷 | 即时 |

## 五、付费深度设计原则

### 5.1 设计原则

1. **不Pay-to-Win（不付费碾压）**
   - PVP使用统一数值BUFF，不使用养成属性
   - 付费加速成长，但零氪也能追上
   - 核心玩法（战役）不以数值卡点强制付费

2. **透明保底机制**
   - 所有抽卡概率公开
   - 保底次数明确显示
   - 累计保底可定向选择

3. **付费即服务**
   - 付费获得更好体验，而非强制需求
   - 皮肤纯外观，不影响数值
   - 战令提供稳定价值感知

4. **价值感知设计**
   - 首充超低价建立锚点
   - 礼包显示"原价vs现价"
   - 限时倒计时创造紧迫感

### 5.2 经济风险控制

| 风险 | 描述 | 控制措施 |
|-----|------|---------|
| 数值膨胀 | 后期武将属性爆炸 | 设置等级上限，赛季重置 |
| 资源溢出 | 老玩家资源过剩 | 持续推出新内容消耗 |
| 通胀失控 | 金币贬值 | 金币消耗与产出平衡 |
| 付费疲劳 | 频繁促销导致反感 | 控制促销频率，提升品质 |

## 六、关键数值建议

### 6.1 数值设计建议

| 参数 | 建议值 | 说明 |
|-----|-------|------|
| 体力上限 | 120点 | 随军衔提升 |
| 体力恢复 | 1点/3分钟 | 满120点停止 |
| 体力消耗/关卡 | 5-15点 | 根据关卡难度 |
| 金币获取/关卡 | 50-500金 | 根据关卡难度 |
| 经验获取/关卡 | 100-1000EXP | 根据关卡难度 |
| 装备掉落率 | 30-50% | BOSS关100% |
| 碎片掉落率 | 10-20% | 精英关更高 |

### 6.2 核心数值公式

**伤害公式：**
```
实际伤害 = 攻击力 × 技能系数 × 兵种克制(1.0/0.75/1.25) × (暴击?1.5:1.0) × (1-护甲减免)
护甲减免 = 护甲 / (护甲 + 100)
```

**怒气获取：**
```
普攻怒气 = 10
受伤怒气 = 5（每次）
击杀怒气 = 20
回合结束怒气 = 5
```

**战斗时长控制：**
```
目标战斗时长 = 2-4分钟/关
敌人数量 = 目标时长 / 平均击杀时间
平均击杀时间 ≈ 3-5秒（随难度调整）
```
