# 游戏战斗系统深化设计

> 本文档深化设计游戏的自动战斗系统、单位AI逻辑、战场机制与技能释放策略。

---

## 一、战斗系统总览

### 1.1 设计理念

战斗系统采用**"战前全策略 + 战中微操作"**的双层模式：

| 阶段 | 玩家行为 | 自动化程度 |
|---|---|---|
| 准备阶段 | 购买、合成、布局、部署 | 0% |
| 部署阶段 | 将单位拖放到棋盘部署区 | 100%（拖放后自动锁定） |
| 战斗阶段 | 手动释放终极技能 | 80%（单位自动行动） |
| 结算阶段 | 查看战报、领取奖励 | 0% |

**参考设计**：
- 循环勇者：战前规划，战中自动
- 王国保卫战：手动技能 + 自动防御塔
- 背包乱闘：纯自动战斗，无手动操作

### 1.2 战斗流程

```
┌─────────────────────────────────────────────────────────────────┐
│                        战 斗 流 程 图                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  【阶段0：战斗开始】                                             │
│      ↓                                                           │
│  双方单位从各自基地出现（我方左侧，敌方右侧）                        │
│      ↓                                                           │

│      ├─ 触发攻击动画/特效                                         │

```

│ 法师          保持中距离       优先攻击聚集区域  范围伤害         │

```

  │ 1. 检查是否死亡                      │
  │    └─ 是：移除单位，结束              │

  ┌─────────────────────────────────────┐
  │ 2. 检查是否有控制效果（眩晕/冻结）    │
  │    └─ 是：跳过本tick，刷新控制倒计时   │
  │    └─ 否：继续                        │

                   ↓
  ┌─────────────────────────────────────┐
  │ 3. 选择攻击目标                      │

  │    ├─ 治疗：最低血量友军              │
  │    └─ 刺客：优先英雄 > 远程单位        │
  └────────────────┬────────────────────┘
                   ↓
  ┌─────────────────────────────────────┐
  │ 4. 检查是否在攻击范围内              │

  │    └─ 否：向目标移动                  │

  │ 5. 执行移动或攻击                    │
  │    ├─ 攻击：计算伤害 → 触发特效       │

### 3.3 仇恨系统

```
仇恨机制（借鉴王国保卫战）：

仇恨值计算：
┌─────────────────────────────────────────────────────────┐
│ 触发事件             仇恨值增加                            │
├─────────────────────────────────────────────────────────┤
│ 对某单位造成伤害      +（造成伤害量 × 10）                 │
│ 进入某单位视野范围    +20（初始仇恨）                     │
│ 嘲讽技能命中         +100（强制嘲讽）                     │
│ 持续未受攻击          每tick -5（仇恨衰减）               │
└─────────────────────────────────────────────────────────┘

仇恨目标选择：
- 坦克单位优先吸引仇恨（嘲讽技能）
- 无坦克时，仇恨值最高的单位被攻击
- 刺客的仇恨目标锁定：英雄 > 治疗 > 远程

嘲讽技能机制：
- 嘲讽范围内的敌人（2格）必须攻击嘲讽者
- 持续时间：2-3秒
- 冷却时间：15-20秒
```

```

### 4.2 手动技能释放界面

```

   - 点击英雄头像或快捷键（Q/E）
   - 弹出技能瞄准界面

3. 瞄准界面（示例：箭雨）：
   ┌────────────────────────────────────┐

   │    敌方聚集区域出现准星提示            │
   │    显示预计伤害范围                    │
   │                                     │
   │    点击确认释放                        │
   │    ESC取消                           │

4. 技能释放：
   - 显示技能动画（1-2秒）
   - 造成预期效果
   - 技能进入冷却，次数-1
```

## 五、伤害计算系统

### 5.1 伤害公式

```
基础伤害公式：

物理伤害 = 攻击方攻击力 × 技能倍率 × 克制系数 × (100 / (100 + 防御方护甲))

魔法伤害 = 攻击方法术强度 × 技能倍率 × 克制系数 × (100 / (100 + 防御方魔抗))

真实伤害 = 攻击方攻击力 × 技能倍率 × 克制系数  （无视护甲/魔抗）

┌─────────────────────────────────────────────────────────────┐
│ 克制关系                                                    │
│ 物理克制：轻型护甲（皮甲） → 伤害+20%                        │
│ 魔法克制：重型护甲（锁甲） → 伤害+20%                        │
│ 轻型护甲被物理克制，被魔法抗性+20%                           │
│ 重型护甲被魔法克制，被物理抗性+20%                           │
│ 魔甲（布袍）：物理抗性+20%，魔法抗性+20%                      │
└─────────────────────────────────────────────────────────────┘
```

## 六、战斗结算系统

### 6.1 胜负判定

```
胜负条件：

胜利条件：
1. 摧毁敌方基地（基础生命值30）
2. 敌方所有单位死亡且无复活能力

失败条件：
1. 我方基地被摧毁（基础生命值30）
2. 战斗中主动退出

特殊情况：
- 超时（3分钟无胜负）：以基地剩余血量%判定，血量高者胜
- 双杀（同时摧毁双方基地）：血量%高者胜
```

战斗结束界面：

┌─────────────────────────────────────────────────────────────┐
│                      第 8 波 · 战 报                         │

│                                                             │
│   胜利 ✓                              用时：1分23秒          │

## 七、战斗节奏设计

### 7.1 战斗时长规划

```
战斗时长设计（基于波次）：

┌─────────────────────────────────────────────────────────────┐
│ 波次范围      预计时长      战斗节奏描述                       │
├─────────────────────────────────────────────────────────────┤
│ 1-5波        30秒-1分钟    试探性，敌人弱，快速通过           │
│ 6-10波       1-2分钟       策略性，敌人有配合，需要部署        │
│ 11-15波      2-3分钟       紧张感，Boss战，需要手动技能        │
│ 16-20波      2-3分钟       高压，每波都是挑战                  │
│ 20波+        3-4分钟       极限挑战，阵容成型后的终极考验       │
└─────────────────────────────────────────────────────────────┘

设计目标：
- 战斗时长不超过4分钟（注意力极限）
- 单局游戏总时长约15-25分钟
- 战斗与准备阶段比例约 1:1（30秒准备 vs 1-2分钟战斗）
```

压力调控手段：
