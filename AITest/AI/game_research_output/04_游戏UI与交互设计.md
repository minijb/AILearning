## 四、交互反馈设计

### 4.1 视觉反馈体系

```
视觉反馈矩阵：

┌─────────────────────────────────────────────────────────────┐

│ 战斗失败               灰暗滤镜 + 裂痕       "..." 低沉音     │
```

### 4.2 动画时长规范

```
动画时长规范（以毫秒为单位）：

物品操作类：
- 物品放置：从拖拽位置→目标位置：200ms ease-out
- 自动吸附：200ms ease-in-out
- 合成粒子：600ms（包含延迟100ms）
- 物品消失（出售）：150ms scale-down + fade

羁绊反馈：
- 羁绊激活：500ms（金光从中心向周围扩散）
- 羁绊升级：800ms（光环扩大 + 数字飞入）
- 羁绊即将激活闪烁：1000ms循环（opacity 0.3→1→0.3）

战斗反馈：
- 伤害数字弹出：300ms float-up + fade
- 暴击数字：400ms + 放大效果
- 治疗数字：300ms float-up（绿色）
- 单位攻击动画：200ms
- 技能释放准备：150ms highlight
- 技能命中：400ms hit-flash on target

界面转场：
- 准备→战斗转场：500ms（屏幕收缩 + 战斗BGM）
- 战斗→准备转场：800ms（棋盘淡出 + 奖励飞入）
- 面板弹出：250ms scale 0.9→1 + fade
- 面板关闭：150ms fade-out

设计原则：
- 所有反馈动画 ≤ 1000ms
- 连续触发（如连续击杀）需有动画队列缓冲
- 低性能设备可关闭部分粒子效果
```

## 五、操作便捷性设计

### 5.1 快捷操作体系

```
快捷键设计：

┌─────────────────────────────────────────────────────────────┐
│ 快捷键         功能                  上下文                  │
├─────────────────────────────────────────────────────────────┤
│ W/A/S/D       移动拖拽中的物品       背包/棋盘              │
│ Q             释放英雄终极技能       战斗阶段                │
│ E             释放英雄第二技能       战斗阶段                │
│ Space         快速进入下一波          准备阶段（计时器归零时）  │
│ R             刷新商店               准备阶段                │
│ B             升级背包               准备阶段                │
│ 1-5           选中棋盘位置1-5        部署阶段                │
│ Tab           切换羁绊/物品详情面板  准备阶段                │
│ Esc           关闭当前面板/取消操作   全局                   │
│ F1            帮助说明               全局                    │
│ Enter         确认/进入战斗          准备阶段结束              │
└─────────────────────────────────────────────────────────────┘

移动端适配（触控手势）：
- 单击：选中/确认
- 长按：显示物品详情
- 拖拽：移动物品
- 双击：快速出售物品
- 两指缩放：调整背包/棋盘视图
- 滑动：滚动商店列表
```

### 5.2 辅助功能

```
辅助功能设计：

1. 自动布局（Auto-Assemble）
   - 一键将当前羁绊最优解自动排列
   - 保留玩家手动调整的空间
   - 适合新手学习羁绊布局

2. 阵容保存/读取
   - 保存当前背包配置为"阵容"
   - 可保存3-5个阵容位
   - 战斗中可快速切换阵容（但需要重新部署）

3. 合成推荐
   - 自动检测背包中满足合成条件的物品
   - 显示合成预览（结果物品预览）
   - 点击确认一键合成

4. 羁绊规划助手
   - 显示当前羁绊完成度
   - 提示缺少哪些单位/物品
   - 推荐当前商店中可购买的羁绊组件

5. 战斗加速（练习模式）
   - 1x / 2x / 3x 速度选项
   - 可跳过战斗动画
   - 适合熟悉战斗节奏
```
