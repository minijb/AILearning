## 核心设计摘要

### 游戏定位
**"背包构筑 + 自动战斗 + 棋盘部署"三合一的中度策略游戏**

### 核心差异化
- **双空间策略**：背包（静态构筑）+ 棋盘（动态部署）
- **空间羁绊**：2x2物理方块激活羁绊，而非简单数量统计
- **轻操作重策略**：只需管理终极技能释放，战斗完全自动

### 目标用户
喜欢背包乱闘/自走棋的空间规划感，但希望有更多操作参与感和策略深度的休闲策略玩家。

### 参考来源
- **Backpack Battles（背包乱闘）**：背包空间限制、套装羁绊、利息机制
- **Kingdom Rush（王国保卫战）**：波次节奏、英雄手动技能、地形策略
- **Age of War（战争进化史）**：升级时机决策、经济流设计
- **Loop Hero（循环勇者）**：空间羁绊的地块协同设计、环境叙事

### 单局时长
- 准备阶段：30秒（固定）
- 战斗阶段：1-3分钟（根据波次）
- 单局总计：15-25分钟
