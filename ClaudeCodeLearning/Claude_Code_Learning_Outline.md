## 学习路径

```
阶段 1-2: 入门基础
  └─ 理解 Claude Code 基本概念，掌握文件操作和终端命令

阶段 3-6: 核心能力
  ├─ 3: Agent 智能体 ──► 并行/串行任务处理
  ├─ 4: Skill 技能 ──► 预定义工作流和自定义 Skill
  ├─ 5: 斜杠命令 ──► 快捷命令系统
  └─ 6: 工具详解 ──► Read/Edit/Write/Glob/Grep/Bash

阶段 7-9: 配置与集成
  ├─ 7: 配置个性化 ──► settings.json、权限、环境变量
  ├─ 8: 高级特性 ──► MCP 服务器、Hooks 机制
  └─ 9: 集成生态 ──► VS Code、GitHub Actions、CI/CD

阶段 10-11: 进阶
  ├─ 10: 最佳实践 ──► 安全、效率、提示词、团队协作
  └─ 11: Rule 规则系统 ──► 强制编码标准、语言特定规则
```

---

## 推荐学习顺序

1. **新手入门** → 阶段 1-2（看完即可开始使用）
2. **效率提升** → 阶段 3（Agent）、阶段 5（命令）
3. **深度定制** → 阶段 7（配置）、阶段 8（MCP/Hooks）
4. **团队协作** → 阶段 10（最佳实践）、阶段 11（Rule）
