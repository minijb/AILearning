## 一、python-development — Python 开发

### 1.1 插件概述

生产级 Python 3.12+ 开发环境，支持 FastAPI、Django、Temporal 等主流框架。

### 1.2 核心 Agent

| Agent | 模型 | 职责 |
|-------|------|------|
| `python-pro` | - | Python 语言专家 |
| `django-pro` | - | Django 框架专家 |
| `fastapi-pro` | - | FastAPI 框架专家 |

### 1.3 5 个专业 Skill

| Skill | 描述 |
|-------|------|
| `async-python-patterns` | asyncio、concurrent.futures、async/await |
| `python-testing-patterns` | pytest、fixtures、mocking |
| `python-packaging` | PyPI 发布、setuptools/pyproject.toml |
| `python-performance-optimization` | cProfile、性能优化 |
| `uv-package-manager` | uv（Astral 的极速 Python 包管理器） |

### 1.4 核心工具

| 命令 | 功能 |
|------|------|
| `python-scaffold` | 项目脚手架生成 |

### 1.5 uv 包管理器

`uv` 是 Astral（ruff 的创建者）开发的极速 Python 包管理器：
- 10-100x 速度提升 vs pip
- 内置 virtualenv 管理
- 兼容 pyproject.toml

### 4.4 K8s Manifest 生成模式

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: <app-name>
  namespace: <namespace>
  labels:
    app: <app-name>
spec:
  replicas: 3
  selector:
    matchLabels:
      app: <app-name>
  template:
    spec:
      containers:
        - name: <container>
          image: <image>:<tag>
          ports:
            - containerPort: <port>
          resources:
            requests:
              cpu: "100m"
              memory: "128Mi"
            limits:
              cpu: "500m"
              memory: "512Mi"
          livenessProbe:
            httpGet:
              path: /health
              port: 8080
          readinessProbe:
            httpGet:
              path: /ready
              port: 8080
```

## 九、PluginEval — 质量评估框架（独立插件）

### 9.1 独立插件

PluginEval 是唯一一个**自包含的质量评估工具**，不依赖其他插件。

### 9.2 三层评估

```
Layer 1: Static Analysis（< 2 秒）
  → 结构检查、反模式检测
  → 免费

Layer 2: LLM Judge（~30 秒）
  → 语义评估
  → 4 次 LLM 调用

Layer 3: Monte Carlo（~2-5 分钟）
  → 50-100 次模拟运行
  → 统计可靠性
```

### 9.3 CLI 命令

```bash
uv run plugin-eval score path/to/skill --depth quick     # Layer 1
uv run plugin-eval score path/to/skill --depth standard # Layer 1+2
uv run plugin-eval certify path/to/skill               # 全量认证
uv run plugin-eval compare skill-a skill-b               # 对比
uv run plugin-eval init plugins/                        # Elo 排名 corpus
```

## 十、核心设计亮点

### 10.1 框架专业化

Python 插件有 3 个 Agent（python-pro + django-pro + fastapi-pro），不是通用 Python Agent，每个框架专家只专注自己的框架。

### 10.2 云平台的 Multi-Agent 协作

cloud-infrastructure 有 6 个 Agent，按专业分工：
- cloud-architect：整体架构
- terraform-specialist：IaC
- kubernetes-architect：容器编排
- network-engineer：网络
- service-mesh-expert：服务网格
- deployment-engineer：部署

### 10.3 Terraform 的独立地位

Terraform Specialist 是独立的 Agent，不是 cloud-architect 的子功能，反映了 IaC 在现代云架构中的核心地位。
