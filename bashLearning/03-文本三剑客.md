## 1. grep — 文本搜索

### 1.1 简介

`grep`（Global Regular Expression Print）是最常用的文本搜索工具，用于在文件或输出中查找匹配指定模式的行。

### 1.2 基本语法

```bash
grep [选项] 模式 [文件...]
```

### 1.3 常用选项

| 选项 | 说明 |
|------|------|
| `-i` | 忽略大小写 |
| `-v` | 反向匹配（不包含模式的行） |
| `-n` | 显示匹配行的行号 |
| `-c` | 统计匹配行的数量 |
| `-r` | 递归搜索目录 |
| `-l` | 只显示包含匹配的文件名 |
| `-w` | 整词匹配 |
| `-o` | 只输出匹配的部分 |
| `-E` | 使用扩展正则表达式（相当于 egrep） |
| `-A n` | 显示匹配行后 n 行 |
| `-B n` | 显示匹配行前 n 行 |

### 1.4 基本示例

```bash
# 在文件中搜索关键词
grep "error" /var/log/syslog

# 忽略大小写搜索
grep -i "warning" log.txt

# 显示行号
grep -n "pattern" file.txt

# 统计匹配数量
grep -c "error" log.txt

# 反向匹配（不包含关键词的行）
grep -v "debug" log.txt

# 递归搜索目录
grep -r "TODO" ./src

# 只显示文件名
grep -l "error" *.log

# 整词匹配
grep -w "the" file.txt
```

### 1.5 使用正则表达式

```bash
# 基本正则：以某字符串开头的行
grep "^root" /etc/passwd

# 以某字符串结尾的行
grep "bash$" /etc/passwd

# 匹配任意单个字符（.）
grep "r..t" /etc/passwd

# 字符类：[ ] 匹配括号内的任意字符
grep "[Aa]pple" file.txt

# 次数匹配：* 匹配零个或多个
grep "go*gle" file.txt    # gogle, google, gooogle...

# 扩展正则（需要 -E）
grep -E "error|warning|critical" log.txt     # 或关系
grep -E "colou?r" file.txt                    # 可选字符
grep -E "go{2,4}" file.txt                     # 2到4次
```

### 1.6 配合管道使用

```bash
# 查看进程
ps aux | grep nginx

# 查看内存使用
free -h | grep Mem

# 查看特定端口
netstat -tuln | grep 8080

# 查找历史命令
history | grep git
```

### 1.7 grep 家族

| 命令 | 说明 |
|------|------|
| `grep` | 标准 grep |
| `egrep` | 等同于 `grep -E`，支持扩展正则 |
| `fgrep` | 快grep，不解析正则，将模式视为固定字符串 |
| `rgrep` | 递归 grep |

## 2. sed — 流编辑器

### 2.1 简介

`sed`（Stream Editor）是一个流编辑器，用于对文本进行替换、删除、插入等操作。它逐行处理文件或输入。

### 2.2 基本语法

```bash
sed [选项] '命令' [文件...]
```

### 2.3 常用选项

| 选项 | 说明 |
|------|------|
| `-n` | 取消默认输出，只显示处理后的行 |
| `-i` | 直接修改文件内容 |
| `-e` | 执行多个编辑命令 |
| `-r` | 使用扩展正则表达式 |

### 2.4 基本替换

```bash
# 格式：s/原内容/新内容/
# 替换每行第一个匹配
sed 's/old/new/' file.txt

# 替换每行所有匹配（加 g）
sed 's/old/new/g' file.txt

# 替换第 N 个匹配
sed 's/old/new/2' file.txt

# 忽略大小写（加 I）
sed 's/old/new/I' file.txt
```

### 2.5 地址定位

```bash
# 指定行号：替换第 3 行
sed '3s/old/new/' file.txt

# 指定范围：替换第 1 到 5 行
sed '1,5s/old/new/' file.txt

# 多行：第 1 行和第 3 行
sed '1s/old/new/; 3s/old/new/' file.txt

# 正则匹配：包含 "error" 的行
sed '/error/s/old/new/' file.txt

# 从某行开始到最后
sed '5,$s/old/new/' file.txt
```

### 2.6 常用编辑命令

```bash
# 删除行
sed '3d' file.txt              # 删除第 3 行
sed '1,5d' file.txt            # 删除 1-5 行
sed '/pattern/d' file.txt      # 删除匹配的行
sed '/^$/d' file.txt           # 删除空行

# 插入行
sed '3i\add new line' file.txt          # 在第 3 行前插入
sed '3a\add new line' file.txt           # 在第 3 行后追加
sed '1i\===header===' file.txt           # 在开头插入

# 修改整行
sed '3c\replaced line' file.txt

# 显示特定行
sed -n '5,10p' file.txt       # 显示 5-10 行
sed -n '10p' file.txt          # 显示第 10 行

# 显示行号（=）
sed -n '/pattern/=' file.txt   # 显示匹配行的行号
```

### 2.7 实战示例

```bash
# 备份并直接修改文件
sed -i.bak 's/old/new/g' file.txt

# 替换多个模式
sed -e 's/foo/bar/g' -e 's/hello/world/g' file.txt

# 组合多个命令
sed -i '/error/d; /^#/d' file.txt

# 交换两个词的位置
sed 's/\(first\)\(second\)/\2\1/' file.txt

# 从管道读取
echo "hello world" | sed 's/world/UNIX/'

# 多行替换（换行符替换）
sed ':a;N;$!ba;s/\n/,/g' file.txt
```

### 2.8 高级用法

```bash
# 使用扩展正则（不用转义）
sed -r 's/(old|new)/replace/g' file.txt

# 反向引用
sed -r 's/([[:digit:]]{4})/[\1]/g' file.txt

# 大小写转换
sed 's/[a-z]/\U&/g' file.txt    # 转大写
sed 's/[A-Z]/\L&/g' file.txt    # 转小写
```

## 3. awk — 报表生成器

### 3.1 简介

`awk` 是一门强大的文本分析语言，最初诞生于 1977 年。它的名字来自三位创始人：Alfred Aho、Peter Weinberger 和 Brian Kernighan。

`awk` 的核心能力：
- 按字段（列）处理文本
- 支持条件判断和循环
- 内置变量和函数
- 适合生成报表

### 3.2 基本语法

```bash
awk '模式 {操作}' [文件...]
```

### 3.3 内置变量

| 变量 | 说明 |
|------|------|
| `$0` | 整行内容 |
| `$1, $2, ... $NF` | 第 1, 2, ... N 个字段 |
| `NF` | 当前行的字段数（Number of Fields） |
| `NR` | 当前行的行号（Number of Record） |
| `FNR` | 当前文件的行号 |
| `FS` | 字段分隔符（默认是空格/制表符） |
| `OFS` | 输出字段分隔符 |
| `ORS` | 输出行分隔符 |
| `RS` | 输入行分隔符 |
| `FILENAME` | 当前文件名 |

### 3.4 基本示例

```bash
# 打印所有行
awk '{print}' file.txt

# 打印第 1 列
awk '{print $1}' file.txt

# 打印多列
awk '{print $1, $3}' file.txt

# 自定义输出分隔符
awk '{print $1 ":" $2}' file.txt

# 显示行号和内容
awk '{print NR, $0}' file.txt

# 打印列数
awk '{print NF, $0}' file.txt
```

### 3.5 设置字段分隔符

```bash
# 按冒号分隔（处理 /etc/passwd）
awk -F: '{print $1, $7}' /etc/passwd

# 等效写法
awk 'BEGIN{FS=":"} {print $1, $7}' /etc/passwd

# 按空格/制表符分隔
awk -F' ' '{print $2}' file.txt

# 多字符分隔符
awk -F',' '{print $1, $3}' file.csv

# 使用正则作为分隔符
awk -F'[/:]' '{print $1}' file.txt
```

### 3.6 条件筛选

```bash
# 打印第一列包含 "error" 的行
awk '/error/ {print}' file.txt

# 打印第一列大于 100 的行
awk '$1 > 100 {print}' file.txt

# 多条件
awk '$1 > 100 && $2 < 50 {print}' file.txt

# 行号条件
awk 'NR==5 {print}' file.txt          # 第 5 行
awk 'NR>=10 && NR<=20 {print}' file.txt  # 10-20 行

# 字符串比较
awk '$1 == "Alice" {print}' file.txt
```

### 3.7 BEGIN 和 END

```bash
# BEGIN：在处理第一行之前执行（常用于初始化）
awk 'BEGIN{print "开始处理"} {print} END{print "处理完成"}' file.txt

# 初始化变量
awk 'BEGIN{sum=0} {sum+=$2} END{print "总和:", sum}' file.txt

# 设置输出格式
awk 'BEGIN{FS=":"; OFS=" | "} {print $1, $3}' /etc/passwd

# 添加表头
awk 'BEGIN{print "姓名", "分数"} {print $1, $2}' score.txt
```

### 3.8 变量与表达式

```bash
# 计算总和
awk '{sum+=$2} END{print sum}' file.txt

# 计算平均值
awk '{sum+=$2} END{print sum/NR}' file.txt

# 计数
awk '/error/{count++} END{print count}' log.txt

# 字符串函数
awk '{print length($0)}' file.txt          # 行长度
awk '{print toupper($1)}' file.txt           # 转大写
awk '{print substr($1, 1, 3)}' file.txt     # 截取子串
awk '{gsub(/old/, "new"); print}' file.txt  # 全局替换
```

### 3.9 数组

```bash
# 统计每个 IP 出现的次数
awk '{ip[$1]++} END{for (i in ip) print i, ip[i]}' access.log

# 统计每个域名的请求数
awk '{domain[$4]++} END{for (d in domain) print d, domain[d]}' access.log

# 二维数组示例
awk '{
    arr[$1,$2]++
} END {
    for (key in arr) {
        split(key, parts, SUBSEP)
        print parts[1], parts[2], arr[key]
    }
}' file.txt
```

### 3.10 函数

```bash
# 内置数学函数
awk 'BEGIN{print sqrt(16), sin(0), rand()}'

# 字符串函数
awk 'BEGIN{
    s = "hello world"
    print length(s)           # 11
    print index(s, "world")    # 7
    print substr(s, 1, 5)      # hello
    print toupper(s)           # HELLO WORLD
    print tolower(s)           # hello world
    gsub("world", "awk", s)    # 替换，返回替换次数
    split("a,b,c", arr, ",")   # 分割到数组
}'

# 自定义函数
awk 'function max(a, b) { return a > b ? a : b }
     {print max($1, $2)}' file.txt
```

### 3.11 流程控制

```bash
# if-else
awk '{
    if ($2 >= 60) {
        grade = "及格"
    } else {
        grade = "不及格"
    }
    print $1, grade
}' score.txt

# 三元运算符
awk '{print ($2 >= 60) ? "pass" : "fail"}' score.txt

# for 循环
awk 'BEGIN{
    for (i=1; i<=5; i++) {
        print i
    }
}'

# while 循环
awk 'BEGIN{
    i=1
    while (i<=5) {
        print i
        i++
    }
}'
```

## 4. 实战综合示例

### 4.1 日志分析

```bash
# 统计 error 出现次数
grep -c "error" app.log

# 提取所有 IP 并统计
awk '{print $1}' access.log | sort | uniq -c | sort -rn | head -10

# 统计每个状态的 HTTP 响应数量
awk '{print $6}' access.log | sort | uniq -c

# 找出错误日志的时间范围
grep "ERROR" app.log | awk '{print $1, $2}' | head -1
grep "ERROR" app.log | awk '{print $1, $2}' | tail -1
```

### 4.2 CSV 数据处理

```bash
# 假设 CSV 文件格式：name,age,city,score

# 打印指定列
awk -F',' '{print $1, $4}' data.csv

# 计算平均分
awk -F',' '$4 ~ /^[0-9]+$/ {sum+=$4; count++} END{print sum/count}' data.csv

# 过滤低于 60 分的学生
awk -F',' '$4 < 60 {print $1, $4}' data.csv

# 按分数排序（需要先添加序号）
awk -F',' '{print NR, $0}' data.csv | sort -k4 -n
```

### 4.3 系统管理

```bash
# 统计磁盘使用
df -h | awk 'NR>1 {print $1, $5}'

# 统计内存使用
free -h | awk 'NR==2{print "使用: "$3", 剩余: "$4}'

# 查看 CPU 信息
cat /proc/cpuinfo | awk '/model name/{print $4, $5, $6; exit}'

# 格式化输出
ps aux | awk '{
    printf "%-10s %-8s %-6s %s\n", $1, $2, $3, $11
}'
```

### 4.4 配置文件处理

```bash
# 提取配置文件中的注释和空行
grep -v '^#' config.conf | grep -v '^$'

# 替换配置文件中的值
sed -i 's/Port 22/Port 2222/' sshd_config

# awk 读取配置项
awk -F= '/^DB_HOST/{print $2}' config.ini

# 批量重命名（sed）
rename 's/\.txt$/.md/' *.txt
```

### 4.5 组合使用

```bash
# 管道组合：awk 提取 -> grep 过滤 -> sed 替换
cat log.txt | awk '{print $4, $5}' | grep "ERROR" | sed 's/^/\[ALERT\] /'

# 从日志中提取 5 分钟内的错误
awk '/'$(date -d '5 minutes ago' '+%Y-%m-%d %H:%M' | cut -d: -f1,2)'/,$0' app.log | grep ERROR

# 生成统计报告
echo "=== 访问统计 ==="
echo "总请求数: $(wc -l < access.log)"
echo "错误请求: $(grep -c ' 5[0-9][0-9] ' access.log)"
echo "IP 统计:"
awk '{print $1}' access.log | sort | uniq -c | sort -rn | head -5
echo "状态码分布:"
awk '{print $6}' access.log | sort | uniq -c | sort -rn
```

### 4.6 常用命令对照表

| 场景 | grep | sed | awk |
|------|------|-----|-----|
| 搜索关键词 | `grep "pattern" file` | — | `/pattern/ {print}` |
| 替换文本 | — | `s/old/new/g` | `{gsub(/old/,"new"); print}` |
| 提取列 | — | — | `{print $1, $3}` |
| 统计行数 | `grep -c "pattern"` | — | `END {print NR}` |
| 删除行 | — | `/pattern/d` | `!/pattern/` |
| 条件筛选 | `grep "pattern"` | — | `$1 > 100 {print}` |
| 格式化输出 | — | — | `printf "%s\t%d\n", $1, $2` |

## 附录：学习建议

1. **从小处入手**：先学会单个命令的基本用法，再尝试组合使用
2. **多用管道**：Linux 的强大之处在于管道组合，三个命令经常一起使用
3. **查看帮助**：`man grep`、`man sed`、`man awk` 是最好的文档
4. **多做练习**：用真实的数据文件练习，从简单到复杂
5. **理解正则**：正则表达式是这三个工具的共同基础
